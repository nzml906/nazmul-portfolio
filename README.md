# Nazmul-portfolio

## Project setup

```
yarn
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```

### Serve locally

```
npm install -g serve
```

```
yarn build
```

```
serve -s dist
```

### Deploy to gh-pages

```
git push origin master
```

```
add vue.config.js
```

```
add deploy.sh
```

```
bash deploy.sh
```
